# 🎬 Movie Sentiment Analysis System

A machine learning project for analyzing movie reviews and classifying them into
**positive, negative, or neutral sentiment** using Natural Language Processing (NLP).

---

## 📌 Overview

This project implements a complete ML pipeline including text preprocessing, TF-IDF feature extraction,
and classification using a Linear Support Vector Machine (LinearSVC).
The system is lightweight, efficient, and provides real-time predictions through a simple interface.

---

## ⚙️ Technologies

Python • Pandas • NLTK • Scikit-learn • TF-IDF • Linear SVM

---

## 🧠 How It Works

The system processes text through several stages:

* Clean and normalize text (lowercase, remove punctuation)
* Tokenization and stopword removal
* Lemmatization
* Convert text to numerical features using TF-IDF
* Train model using LinearSVC
* Predict sentiment for new input

📌 Based on the project methodology, the model uses preprocessing + TF-IDF + LinearSVC to achieve efficient sentiment classification
